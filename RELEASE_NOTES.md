## 🎉 ClipPal v1.0 发布

### ✨ 新功能

- **开机自动启动** - 在设置中开启后，系统启动时自动运行 ClipPal
- **全新的关于页面** - 采用苹果风格设计，更加简洁专业
- **应用图标优化** - 菜单栏和关于页都使用 App Icon，并添加阴影效果

### 📝 功能特性

- 📋 剪贴板历史记录管理
- 🔍 智能搜索支持
- 📌 固定常用项目
- ⌨️ 全局快捷键 ⌘⇧V
- 🖼️ 支持文本、图片、文件等多种类型
- ⚙️ 开机自启动设置
- 💾 数据持久化存储

### 💻 系统要求

- macOS 13.0 或更高版本
- Apple Silicon 或 Intel Mac

### 📦 安装说明

1. 下载 `ClipPal-1.0-macOS.dmg`
2. 双击打开 DMG，将 ClipPal 拖拽到 Applications 文件夹
3. 首次运行时，按住 Control 键点击应用图标，选择"打开"
4. 如需使用全局快捷键，请在系统设置中授予辅助功能权限
